# 🏔️ Everest MC

에베레스트 MC에 오신 것을 환영합니다!  
저희 서버는 모험과 창작을 모두 즐길 수 있는 멀티플레이 환경을 목표로 운영되고 있습니다.

짜릿한 생존을 경험할 수 있는 야생 서버, 그리고 자유롭게 아이디어를 펼칠 수 있는 공간까지 —  
플레이 스타일에 상관없이 누구나 즐길 수 있도록 준비했습니다.

## 🌐 크로스 플랫폼 지원

에베레스트 MC는 `Java Edition`과 `Bedrock Edition` 모두 접속 가능합니다.  
PC와 모바일, 어떤 환경에서도 자유롭게 접속해 보세요!

## 🧭 진짜 “야생”에서 시작하는 생존

처음 접속하면 정해진 스폰 없이 완전히 새로운 세계에 떨어지게 됩니다.

- 예측 불가능한 시작 위치  
- 플레이어 간 자유로운 상호작용  
- 약탈 가능 (Raid Allowed)

진정한 의미의 생존을 경험하고 싶은 분들을 기다립니다.
